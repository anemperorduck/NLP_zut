# AI 大模型开发专家导师 - Ollama 对话系统

基于 Ollama 和 DeepSeek-R1 模型的 AI 大模型开发专家导师对话系统，专门为计算机专业大三学生提供大模型开发领域的学习指导。

## 项目简介
·
本项目是一个专门针对"人工智能大模型开发"领域的智能问答系统，通过 Ollama 平台调用 DeepSeek-R1:7b 模型，为学生提供专业的大模型开发知识讲解、代码示例和学习指导。

### 核心功能

- ✅ **专业领域问答**：专注于大模型开发相关的理论和技术问题
- ✅ **多轮对话支持**：保持对话上下文，实现连续交互
- ✅ **流式输出**：实时显示模型生成内容，提升用户体验
- ✅ **严格领域限制**：自动识别并拒绝与大模型开发无关的问题
- ✅ **结构化输出**：按照问题分析、核心解答、代码示例、延伸思考的格式组织内容

## 🚀 快速开始

### 环境要求

- Python 3.8+
- Ollama 服务
- `deepseek-r1:7b` 模型

### 安装步骤

1. **安装 Ollama**
   
   访问 [Ollama 官网](https://ollama.ai) 下载并安装 Ollama

2. **拉取模型**
   
   ```bash
   ollama pull deepseek-r1:7b
   ```

3. **安装 Python 依赖**
   
   ```bash
   pip install ollama
   ```

4. **运行程序**
   
   ```bash
   python ollama_tutor_chat.py
   ```

## 💡 使用示例

启动程序后，您可以直接提出与大模型开发相关的问题，例如：

```
你：请解释一下 Transformer 中的自注意力机制是如何工作的？

导师：[详细的理论讲解和代码示例]
```

### 输入命令

- 直接输入问题进行交流
- 输入 `quit`、`exit` 或 `退出` 结束对话

## 系统特色

### Role 设定

系统扮演一位拥有丰富学术界与工业界经验的"人工智能大模型开发"专家导师，具备以下能力：

- **深度理论解析**：讲解大模型背后的数学原理和模型架构
- **工程实践指导**：提供 PyTorch/Hugging Face 框架的代码示例
- **学术前沿追踪**：熟悉最新论文进展，引导阅读经典文献
- **学习路径规划**：定制适合大三学生的学习路线

### 约束机制

系统严格遵守领域限制，只回答与"人工智能大模型开发"直接相关的问题，包括：
- 算法原理
- 数据处理
- 算力优化
- 框架使用
- 学术论文

对于无关问题，系统会统一拒绝回答。

## 配置说明

### 模型配置

当前使用的模型为 `deepseek-r1:7b`，如需更换模型，可修改 `ollama_tutor_chat.py` 中的 `MODEL_NAME` 常量：

```python
MODEL_NAME = "deepseek-r1:7b"  # 修改为其他支持的模型
```

### System Prompt

系统的核心提示词定义了角色设定、核心能力和输出格式，可根据需求在 `SYSTEM_PROMPT` 变量中调整。

## 注意事项

1. 确保 Ollama 服务已启动
2. 确保已下载对应的模型文件
3. 首次运行时模型下载可能需要较长时间
4. 如遇 `ResponseError`，请检查 Ollama 服务状态

## 故障排除

### 常见问题

**Q: 出现 `ResponseError` 错误**
- 确认 Ollama 服务已启动
- 检查模型 `deepseek-r1:7b` 是否已拉取
- 查看 Ollama 日志排查具体问题

**Q: 模型响应速度慢**
- 检查 GPU/CPU 资源使用情况
- 考虑使用更小的模型版本
- 关闭其他占用资源的程序
